## PROJECT OVERVIEW

FHI 360 / EpiC Project seeks to support Maharashtra State AIDS Control Society (MSACS) in achieving India's 95-95-99 HIV cascade targets through strengthened Strategic Information systems. This proposal outlines SLA Softtech's comprehensive approach to design and implement four integrated automated digital solutions.

**Project Scope:**
1. Automated State Sankalak Report System
2. District HIV Epidemic Profiling (DEP) Dashboards
3. ART Data Deduplication Platform (Interstate)
4. KPI/PMS Factsheet and Gap Analysis Tool

**Project Duration:** 25 May - 25 September 2026 (4 months, 18 weeks)
**Project Investment:** ₹38,64,500 (including 18% GST)
